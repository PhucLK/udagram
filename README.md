# Udagram

## Hosting link
The link to the application can be seen [here](http://udagram-hosting.s3-website.us-east-2.amazonaws.com/home).

## Installation

Run `npm install` to download the necessary dependencies.

## ENV file set up
- AWS_ACCESS_KEY_ID
- AWS_DEFAULT_REGION
- AWS_SECRET_ACCESS_KEY
- JWT_SECRET
- POSTGRES_DB
- POSTGRES_HOST
- POSTGRES_PASSWORD
- POSTGRES_PORT
- POSTGRES_USERNAME
- PORT 
- URL
